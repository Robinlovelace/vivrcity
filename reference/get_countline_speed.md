# Get Countline Speeds

Get Countline Speeds

## Usage

``` r
get_countline_speed(countline_ids, from, to, time_bucket = "24h")
```

## Arguments

- countline_ids:

  Vector of countline IDs.

- from:

  Start timestamp.

- to:

  End timestamp.

- time_bucket:

  Time bucket size (e.g. "1h", "5m"). Defaults to "24h".

## Value

Data frame of speeds.
